# .websidian

Configuration for **Websidian**, created automatically the first time this
vault was opened — the vault's own settings directory, versioned alongside the
notes it styles.

- `settings.json` — app behavior (e.g. beta features)
- `appearance.json` — default CSS theme + base color scheme
- `keybindings.json` — keyboard shortcuts (`action → "Mod+Alt+Key"` chords;
  `Mod` is ⌘ on macOS / Ctrl elsewhere, an empty array disables an action)

Edit these files to change the defaults. You can also add your own CSS theme
under `themes/<Name>/` (a `manifest.json` + `theme.css`).
